## Some of my projects

### [DB-Talk](https://github.com/KAMAL-02/DB-Talk)
**AI-Powered Database Query Assistant**
Convert natural language into executable SQL and MongoDB queries. Ask questions in plain English, get real query results.
- Built with **Next.js, TypeScript, Fastify, PostgreSQL, Redis, Drizzle ORM, Gemini API**
- Handles tool use, context management, and multi-database orchestration
