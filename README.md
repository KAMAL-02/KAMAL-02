### [Payout Engine](https://github.com/KAMAL-02/Payout-engine)
**Payout Processing System** : 
Simulates real-world money movement using append-only ledger, idempotent APIs, row-level locking, and async processing with retries.
- Built with **Django, DRF, PostgreSQL, Redis, Celery, React, TypeScript**
- Append-only ledger architecture for accurate balance computation
- Idempotent payout handling with request deduplication and response replay
- Concurrency-safe fund reservation using database transactions and row-level locks
- Async payout processing with retry and recovery workflows using Celery + Redis
